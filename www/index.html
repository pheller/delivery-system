Reloaded

The Prodigy “Interactive Personal Service” launched in 1984 as a joint venture between IBM and Sears, was rebranded as “Prodigy Classic” as the company transitioned to more standards-based Internet service, and was retired in 1999.

Prodigy Reloaded continues the efforts of the Prodigy Preservation Project, which seeks to preserve Prodigy Classic content by collecting artifacts, especially the cache files used by the Prodigy client, from which original content is recovered. Prodigy Reloaded makes use of this discovered content to present an API compatible Prodigy Classic server to which the original client can connect.
